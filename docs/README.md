# FridgeFinder Notifications API Documentation

Interactive API documentation hosted via GitHub Pages.

## View Documentation

Visit: https://fridgefinder.github.io/CFM_Notification/

## Local Development

To view the documentation locally:

1. Install a local web server (e.g., Python's http.server)
2. Navigate to the docs directory
3. Run: `python3 -m http.server 8000`
4. Open: http://localhost:8000

## Updating Documentation

1. Edit `docs/openapi.yaml`
2. Commit and push changes
3. GitHub Pages will automatically update

## Files

- `index.html` - Swagger UI interface
- `openapi.yaml` - OpenAPI 3.0 specification
